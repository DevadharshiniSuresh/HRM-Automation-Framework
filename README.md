# Selenium Hybrid Automation Framework

 Project Overview
This project is a Hybrid Test Automation Framework developed using Selenium, Java, TestNG, and Maven.

The framework is designed using Page Object Model (POM) with reusable utilities, configuration management, and reporting features to ensure maintainability, scalability, and readability.

Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Git & GitHub
